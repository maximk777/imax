UI решения

  • 4 экрана для MVP: Onboarding, Home, Chat, New Chat (+ Settings позже)
  • Статусы сообщений: ✓ sent, ✓✓ delivered, ✓✓ read
  • Онлайн-индикатор: зелёный для подключённых пиров
  • Dracula-подобная тема: тёмная, контрастная (можно менять)
  • Dioxus Signals: реактивное состояние, UI перерисовывается автоматически
